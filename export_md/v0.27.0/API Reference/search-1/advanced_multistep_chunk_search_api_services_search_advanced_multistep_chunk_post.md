excerpt: "Performs a vector hybrid search and then an advanced chunk search.\n\nChunk Search allows one to search through chunks inside a document. The major difference between chunk search and normal search in Vector AI is that it relies on the _chunkvector_ field.\nChunk Vector Search. Search with a multiple chunkvectors for the most similar documents.\nChunk search also supports filtering to only search through filtered results and facets to get the overview of products available when a minimum score is set.\n\nExample 1 (Hybrid chunk search):\n\n    chunk_query1= {\n        \"chunk\" : \"some.test\",\n        \"queries\" : [\n            {\"vector\" : vec1, \"fields\": {\"some.test.some_chunkvector_\":1},\n            \"traditional_query\" : {\"text\":\"python\", \"fields\" : [\"some.test.test_words\"], \"traditional_weight\": 0.3},\n            \"metric\" : \"cosine\"},\n            {\"vector\" : vec, \"fields\": [\"some.test.tt.some_other_chunkvector_\"],\n            \"traditional_query\" : {\"text\":\"jumble\", \"fields\" : [\"some.test.test_words\"], \"traditional_weight\": 0.3},\n            \"metric\" : \"cosine\"},\n        ]\n    }\n\nExample 2 (combines normal vector search with chunk search):\n\n    chunk_query1= {\n    \"queries\" : [\n        {\n            \"queries\": [\n                {\n                    \"vector\": vec1,\n                    \"fields\": {\n                        \"some.test.some_chunkvector_\": 0.9\n                    },\n                    \"traditional_query\": {\n                        \"text\": \"python\",\n                        \"fields\": [\n                            \"some.test.test_words\"\n                        ],\n                        \"traditional_weight\": 0.3\n                    },\n                    \"metric\": \"cosine\"\n                }\n            ],\n            \"chunk\": \"some.test\",\n        },\n        {\n            \"vector\" : vec,\n            \"fields\": {\n                \".some_vector_\" : 0.1},\n                \"metric\" : \"cosine\"\n                },\n        ]\n    }"
